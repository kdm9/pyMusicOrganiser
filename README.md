pyMusicOrganiser
================

[![Build Status](https://travis-ci.org/kdmurray91/pyMusicOrganiser.png)](https://travis-ci.org/kdmurray91/pyMusicOrganiser)


A commandline program ``pymo`` to reorganise music collections based on tags.


Installation
============
